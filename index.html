Pensamento Computacional

Pensamento Computacional é uma disciplina introduzida no Novo Ensino Médio e tem a intenção de iniciar os estudantes ao mundo da programação.

Nessa disciplina os estudantes são convidados a desenvolver jogos utilizando a plataforma do MIT, Scratch. Além disso, também desenvolvem alguns programas utilizando a linguagem de programação JavaScript, bem como sites utilizando HTML e CSS.

Nesta página você tem acesso a alguns dos trabalhos desenvolvidos com os estudantes.
